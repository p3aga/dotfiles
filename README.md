# Dotfiles

Este repositorio contém uma configuração modular de ambiente para Linux, gerenciada com chezmoi.

## Requisitos

- chezmoi
- git
- curl
- tar
- zsh

## O que é instalado/configurado

### 1. Homebrew

É instalado o homebrew para gerenciamento de CLIs/TUIs.

### 2. Fontes

Maple Mono Nerd Font (patch de [maple-font](https://github.com/subframe7536/maple-font)).

### 3. Zsh

Shell modularmente configurado com `romkatv/powerlevel10k`, `zsh-users/zsh-completions`, `zsh-users/zsh-syntax-highlighting`, `zsh-users/zsh-autosuggestions` e `Aloxaf/fzf-tab`, além do `zinit` para gerenciamento de plugins.


### 4. Mise

O mise é utilizado para gerenciamento de versões de linguagens(substituto ao `asdf`).


### 5. Neovim

Utilizada a configuração de Neovim de `p3aga/nvim.lua`.
